Estructuras de control cíclicas for, while, do-while

En programación, las estructuras de control permiten modificar el flujo de ejecución de las instrucciones de un programa. Con las estructuras de control se puede: De acuerdo con una condición, ejecutar un grupo u otro de sentencias (If-Else) De acuerdo con el valor de una variable, ejecutar un grupo u otro de sentencias (Switch) Ejecutar un grupo de sentencias mientras se cumpla una condición (While) Ejecutar un grupo de sentencias hasta que se cumpla una condición (for…in) Ejecutar un grupo de sentencias un número determinado de veces (For)
